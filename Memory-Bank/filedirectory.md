# File Directory: Shannon's Softball Portfolio

## Root Directory
- `index.html` - Main website page with all sections
- `README.md` - Project documentation and setup instructions

## CSS Directory (`/css/`)
- `style.css` - Main stylesheet with core styling
- `responsive.css` - Mobile-specific responsive styles

## JavaScript Directory (`/js/`)
- `main.js` - Core functionality and page interactions
- `stats.js` - Statistics dashboard functionality
- `video-player.js` - Video gallery and player controls

## Images Directory (`/images/`)

- `action-shots/` - Directory for action photos

- `batting/` - Batting highlight videos
- `fielding/` - Fielding highlight videos

- `thumbnails/` - Video thumbnail images

- `recruiting-packet.pdf` - Downloadable recruiting packet

## Memory Bank Directory (`/Memory-Bank/`)
- `projectbrief.md` - Project overview and objectives
